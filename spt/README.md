# Spotfynder (Frontend) & API (Backend)

Проект **Spotfynder**. Репозиторий содержит как клиентскую (Frontend), так и серверную (Backend) часть.

## Как запустить проект локально

Для корректной работы приложения вам потребуется открыть **два отдельных терминала** — один для бэкенда, другой для фронтенда.

### 1. Запуск Backend-сервера (Python / FastAPI)

Откройте первый терминал и выполните следующие команды:

```bash
# 1. Перейдите в директорию бэкенда:
cd /mnt/storage/projects/spt/SPSPSP/Backend

# 2. Активируйте виртуальное окружение:
source .venv/bin/activate

# 3. Запустите API-сервер:
uvicorn app.main:app --host 127.0.0.1 --port 10000 --reload
```

*Сервер будет запущен на порту `10000`. Приложение Flutter автоматически настроено на отправку запросов именно по этому адресу.*

### 2. Запуск Frontend-приложения (Flutter Web)

Откройте **второй** терминал и выполните следующее:

```bash
# 1. Перейдите в директорию клиента:
cd /mnt/storage/projects/spt/SPSPSP/spt

# 2. Запустите Flutter-сервер:
flutter run -d web-server --web-hostname 0.0.0.0 --web-port 8080
```

После выполнения команды в терминале появится ссылка (вида `http://localhost:8080` или `http://0.0.0.0:8080`). Кликните по ней, чтобы открыть приложение в браузере.

### Примечания при разработке (Flutter):
При запущенном через консоль `flutter run` сервере, находясь в этом терминале, можно использовать горячие клавиши:
* `r` — **Hot Reload** (Мгновенное обновление интерфейса без сброса).
* `R` — **Hot Restart** (Полный перезапуск приложения в браузере).
* `q` — Завершить работу (`Quit`).
